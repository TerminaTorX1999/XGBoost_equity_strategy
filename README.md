# XGBoost-Based Equity Directional Trading Strategy

This repository contains a Python implementation of a binary classification trading strategy using XGBoost.

## Features
- 20-day EMA, RSI, and lagged returns as features
- Signal generation and backtesting framework
- Confusion matrix and directional accuracy evaluation

## Usage
1. Install dependencies: `pandas`, `numpy`, `yfinance`, `scikit-learn`, `xgboost`, `matplotlib`
2. Run `xgboost_directional_strategy.py`
